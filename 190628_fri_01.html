DOM

Document Object Model

HTML은 트리구조를 가진 객체이다.

이 객체를 JavaScript를 이용해 어떻게 접근할 수 있을까?

그리고 구성요소(Element)의 추가, 수정 ,삭제는 어떻게 해?

ELEMENT vs. NODE

  • Element는 Node에 속해있다
  • Text(content)는 Node이나, Element는 아니다.

어떻게 HTML element를 JavaScript를 이용해 가져올 수 있을까?

  • tag name 이용 : getElementsByTagName
  • id 속성 이용 : getElementById
  • class name 이용 : getElementByClassName
  • selector 이용 : querySelector / querySelectorAll

<script> tag

대표적으로 사용되는 위치

  1. <head>태그 안쪽 : DOM이 로딩되지 않은상태에서 JavaScript 실행 / 일반적으로 JavaScript 라이브러리를 추가할 때 사용
<head>
  <script src="main.js"></script>
</head>

2. <body>태그가 닫히기 직전 : DOM이 로딩되고 난 후 JavaScript실행 / DOM을 직접 다루는 경우에 사용

<body>
  <div id="greeting">Welcom</div>
  <!--생략-->
  <script src="main.js"></script>
</body>
